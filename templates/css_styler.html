joysticktv overlay CSS Styler

How to use:
1. Select a "Vanilla" scene in Joystick.TV Scenes, and add it to your OBS as a Browser Source according to JSTV's instructions.
2. Use this tool to design your look.
3. Copy the CSS below.
4. In OBS, right-click the Browser Source → Properties → Paste into "Custom CSS".
5. Click the "Refresh cache of current page" button at the bottom of this Properties window. → OK.
